What is this

This began as the solution for one of the projects in The Odin Project to create an "Etch a Sketch" website with a grid you could draw on.
Later when taking the CS50 course i improved the website during the html/css section. That section assignment involved creating a website that fit certain criteria:

  • Has at least 4 html pages
  • Utilizes one or more features from Bootstrap
  • Contains custom css styles
  • Contains custom javascript code
I reused the etch a sketch project and added extra pages to fit the spec, including a functioning gallery one could use to save a canvas and share it.

About

This project ended up growing beyond the original scope of simply being a homepage with 4 html files to get over an annoying part of the course by reusing an old project, so now it sits here as part of the portifolio, available on Github.

This is Canvas50

A simple but functional etch a sketch with a customizable grid and the ability to save drawings and share them.
